# Shark

An artificial intelligence made with HaxeFlixel.

## About

Shark is an AI system built on top of HaxeFlixel, designed to be integrated into games and interactive projects. It provides a lightweight foundation for implementing intelligent behavior directly within a HaxeFlixel game loop.

## Features

- Built entirely with Haxe and HaxeFlixel
- Designed for integration into existing HaxeFlixel projects
- Lightweight and extensible architecture

## Requirements

- [Haxe](https://haxe.org/)
- [HaxeFlixel](https://haxeflixel.com/)

## Installation

Clone the repository:

```bash
git clone https://github.com/Brenninho123/Shark.git
```

Add the library to your project according to your Haxe/HaxeFlixel setup.
